# this-door-bot
A simple telegram bot returns this day in history  
Based on Spring Boot and [rubenlagus' api](https://github.com/rubenlagus/TelegramBots/tree/master/telegrambots-spring-boot-starter), data from [inside-door-restful-api](https://github.com/h53/inside-door-restful-api)  
There is a tutorial recommand to you, [Writing Telegram Bots on Java](https://monsterdeveloper.gitbooks.io/writing-telegram-bots-on-java/content/)  
You can try this bot via https://t.me/ThisDoor_bot
